# OpenAbyss
Open source script for Powbot that crafts runes through the abyss

# Features

- Multiple methods of crafting
  - Chronicle
  - House teleport (Runes)
- Runes supported
  - Air
  - Cosmic
  - Earth
  - Fire
  - Nature
- Energy restoration supported
  - Energy potions
  - Strange fruit
